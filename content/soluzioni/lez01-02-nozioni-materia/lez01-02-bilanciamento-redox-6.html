1 Reazione 1: Ag2O + H2O2 -> Ag + H2O + O2

Esame 8 - Quesito 4: Bilanciamento Reazioni Redox

Testo del quesito:
Completare e bilanciare le seguenti reazioni:
1) Ag2O + H2O2 = Ag + H2O + O2
2) Si + NaOH = Na4SiO4 + H2
3) Fe + HCl = FeCl2 + H2

Reazione 1: Ag2O + H2O2 -> Ag + H2O + O2

Iniziamo analizzando questa reazione. Dobbiamo prima capire cosa si ossida e cosa si riduce. Per farlo, assegniamo i numeri di ossidazione a tutti gli elementi.

Assegnazione numeri di ossidazione

Nei reagenti:

  • Ag2O: L'ossigeno ha numero di ossidazione -2. Per bilanciare, l'argento deve avere +1. Quindi Ag ha n.o. = +1
  • H2O2 (acqua ossigenata): L'idrogeno ha n.o. = +1. L'ossigeno in H2O2 e' un caso particolare: ha n.o. = -1 (non -2 come di solito!)

Nei prodotti:

  • Ag metallico: n.o. = 0 (elemento libero)
  • H2O: H = +1, O = -2
  • O2: n.o. = 0 (elemento libero)

Identificazione ossidazione e riduzione

Argento: passa da +1 (in Ag2O) a 0 (in Ag) -> RIDUZIONE (acquista 1 elettrone per atomo)

Ossigeno di H2O2: passa da -1 a 0 (in O2) -> OSSIDAZIONE (perde 1 elettrone per atomo)
Attenzione! L'ossigeno di H2O2 non diventa tutto O2. Una parte va a formare H2O (dove O ha n.o. -2). Ma per la parte che diventa O2, c'e' ossidazione.

Scrittura delle semireazioni

Semireazione di riduzione (Ag+1 -> Ag0):

Ag+ + 1e- -> Ag

Ma nell'Ag2O abbiamo 2 atomi di argento, quindi:

Ag2O + 2e- -> 2Ag + O2-

L'ossido libera ione ossido. In ambiente acquoso:

Ag2O + 2H+ + 2e- -> 2Ag + H2O

Semireazione di ossidazione (O-1 in H2O2 -> O0 in O2):

H2O2 -> O2 + 2H+ + 2e-

Qui 2 atomi di O (ciascuno con -1) perdono 1 elettrone ciascuno, diventando O2 (n.o. = 0).

Bilanciamento delle semireazioni

Le due semireazioni sono gia' bilanciate in termini di elettroni (entrambe coinvolgono 2 elettroni):

Riduzione: Ag2O + 2H+ + 2e- -> 2Ag + H2O
Ossidazione: H2O2 -> O2 + 2H+ + 2e-

Somma delle semireazioni

Sommando le due semireazioni (gli elettroni si cancellano, e anche i 2H+ si cancellano):

Ag2O + H2O2 -> 2Ag + H2O + O2

Verifica del bilanciamento

Contiamo gli atomi da entrambi i lati:

  • Ag: 2 a sinistra, 2 a destra ✓
  • O: 1 (da Ag2O) + 2 (da H2O2) = 3 a sinistra; 1 (da H2O) + 2 (da O2) = 3 a destra ✓
  • H: 2 a sinistra (da H2O2), 2 a destra (da H2O) ✓
Reazione 1 bilanciata:
Ag2O + H2O2 -> 2Ag + H2O + O2
2 Reazione 2: Si + NaOH -> Na4SiO4 + H2

Reazione 2: Si + NaOH -> Na4SiO4 + H2

Questa reazione avviene in ambiente basico (c'e' NaOH). Il silicio reagisce con la soda per formare un silicato e idrogeno gassoso.

Assegnazione numeri di ossidazione

Nei reagenti:

  • Si elementare: n.o. = 0
  • NaOH: Na = +1, O = -2, H = +1

Nei prodotti:

  • Na4SiO4: Na = +1, O = -2. Per il silicio: 4(+1) + Si + 4(-2) = 0, quindi Si = +4
  • H2: n.o. = 0

Identificazione ossidazione e riduzione

Silicio: passa da 0 a +4 -> OSSIDAZIONE (perde 4 elettroni)

Idrogeno: passa da +1 (in NaOH) a 0 (in H2) -> RIDUZIONE (acquista 1 elettrone per atomo)

Scrittura delle semireazioni

Semireazione di ossidazione (Si0 -> Si+4):

In ambiente basico, usiamo OH- e H2O per bilanciare:

Si + 8OH- -> SiO4(4-) + 4H2O + 4e-

Verifica: Si passa da 0 a +4, quindi perde 4 elettroni. Corretto!

Semireazione di riduzione (H+1 -> H0):

2H2O + 2e- -> H2 + 2OH-

(Questa e' la forma della riduzione dell'acqua in ambiente basico)

Bilanciamento degli elettroni

La semireazione di ossidazione produce 4 elettroni, quella di riduzione ne consuma 2. Dobbiamo moltiplicare la riduzione per 2:

Ossidazione: Si + 8OH- -> SiO4(4-) + 4H2O + 4e-
Riduzione (x2): 4H2O + 4e- -> 2H2 + 4OH-

Somma delle semireazioni

Sommando:

Si + 8OH- + 4H2O -> SiO4(4-) + 4H2O + 2H2 + 4OH-

Semplificando (4H2O si cancellano, e 4OH- si cancellano):

Si + 4OH- -> SiO4(4-) + 2H2

Ora aggiungiamo i 4 ioni Na+ per avere Na4SiO4:

Si + 4NaOH -> Na4SiO4 + 2H2

Verifica del bilanciamento

  • Si: 1 a sinistra, 1 a destra ✓
  • Na: 4 a sinistra, 4 a destra ✓
  • O: 4 a sinistra (da 4 NaOH), 4 a destra (da Na4SiO4) ✓
  • H: 4 a sinistra (da 4 NaOH), 4 a destra (da 2 H2) ✓
Reazione 2 bilanciata:
Si + 4NaOH -> Na4SiO4 + 2H2
3 Reazione 3: Fe + HCl -> FeCl2 + H2

Reazione 3: Fe + HCl -> FeCl2 + H2

Questa e' la reazione classica tra un metallo e un acido. Il ferro metallico reagisce con acido cloridrico per dare cloruro ferroso e idrogeno gassoso.

Assegnazione numeri di ossidazione

Nei reagenti:

  • Fe metallico: n.o. = 0
  • HCl: H = +1, Cl = -1

Nei prodotti:

  • FeCl2: Cl = -1, quindi Fe = +2
  • H2: n.o. = 0

Identificazione ossidazione e riduzione

Ferro: passa da 0 a +2 -> OSSIDAZIONE (perde 2 elettroni)

Idrogeno: passa da +1 a 0 -> RIDUZIONE (acquista 1 elettrone per atomo)

Scrittura delle semireazioni

Semireazione di ossidazione (Fe0 -> Fe2+):

Fe -> Fe2+ + 2e-

Semireazione di riduzione (H+ -> H0):

2H+ + 2e- -> H2

(Servono 2 ioni H+ per formare una molecola di H2, e ogni H+ acquista 1 elettrone)

Bilanciamento degli elettroni

Entrambe le semireazioni coinvolgono 2 elettroni, quindi sono gia' bilanciate:

Ossidazione: Fe -> Fe2+ + 2e-
Riduzione: 2H+ + 2e- -> H2

Somma delle semireazioni

Sommando (gli elettroni si cancellano):

Fe + 2H+ -> Fe2+ + H2

Ora aggiungiamo i cloruri (2 Cl- per ogni Fe2+ e 1 Cl- per ogni H+):

Fe + 2HCl -> FeCl2 + H2

Verifica del bilanciamento

  • Fe: 1 a sinistra, 1 a destra ✓
  • H: 2 a sinistra (da 2 HCl), 2 a destra (da H2) ✓
  • Cl: 2 a sinistra, 2 a destra ✓
Reazione 3 bilanciata:
Fe + 2HCl -> FeCl2 + H2
4 Riepilogo delle tre reazioni bilanciate

Riepilogo delle tre reazioni bilanciate

1) Ag2O + H2O2 -> 2Ag + H2O + O2

2) Si + 4NaOH -> Na4SiO4 + 2H2

3) Fe + 2HCl -> FeCl2 + H2

Ricorda: nel bilanciamento redox con il metodo delle semireazioni, la chiave e' sempre identificare correttamente cosa si ossida e cosa si riduce, poi bilanciare gli elettroni persi e acquistati!
