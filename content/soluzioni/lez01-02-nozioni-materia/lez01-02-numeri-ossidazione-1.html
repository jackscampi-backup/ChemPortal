Esame 8 - Quesito 2: Numeri di Ossidazione

TESTO DEL QUESITO:

Assegnare ad ogni atomo il numero di ossidazione nelle seguenti specie:
  1. C3H6O (acetone o propanale)
  2. H2O2 (acqua ossigenata)
  3. Idrogenofosfato di Magnesio (MgHPO4)
  4. Acido lattico: CH3-CHOH-COOH
  5. Serina (amminoacido): H2N-CH(CH2OH)-COOH
  6. Alanina (amminoacido): H3C-CH(NH2)-COOH

Premessa: Le Regole per Assegnare i Numeri di Ossidazione

Prima di affrontare gli esercizi, fermati un attimo a capire cosa sono i numeri di ossidazione e come si assegnano. Il numero di ossidazione (n.o.) rappresenta la carica che un atomo avrebbe se tutti i legami fossero ionici, cioe se gli elettroni di legame appartenessero completamente all'atomo piu elettronegativo.

REGOLE FONDAMENTALI per assegnare i numeri di ossidazione:
  1. Elementi allo stato elementare: n.o. = 0 (es. Fe, O2, N2, H2)
  2. Ioni monoatomici: n.o. = carica dello ione (es. Na+ ha n.o. = +1, Cl- ha n.o. = -1)
  3. Idrogeno: n.o. = +1 (quasi sempre), tranne negli idruri metallici dove e -1
  4. Ossigeno: n.o. = -2 (quasi sempre), tranne nei perossidi (-1) e nel fluoruro di ossigeno (+2)
  5. Fluoro: sempre n.o. = -1 (e il piu elettronegativo)
  6. Metalli alcalini (Gruppo 1): sempre n.o. = +1
  7. Metalli alcalino-terrosi (Gruppo 2): sempre n.o. = +2
  8. La somma dei n.o. in una molecola neutra deve essere ZERO
  9. La somma dei n.o. in uno ione poliatomico deve essere uguale alla carica dello ione
ATTENZIONE - Composti organici:
Nei composti organici, ogni atomo di carbonio puo avere un diverso numero di ossidazione a seconda dei suoi legami! Non esiste "il" numero di ossidazione del carbonio nella molecola, ma bisogna calcolarlo per ciascun atomo di C separatamente, considerando:
  • Legame C-H: il carbonio "prende" l'elettrone (H e meno elettronegativo), quindi ogni H porta +1 al n.o. del C
  • Legame C-C: gli elettroni si dividono equamente (stessa elettronegativita), contributo = 0
  • Legame C-O, C-N, C-Cl: il carbonio "perde" gli elettroni (O, N, Cl piu elettronegativi), ogni legame porta -1 al n.o. del C (per legame doppio -2)
2 Esercizio 1: C3H6O (acetone o propanale)

Esercizio 1: C3H6O (acetone o propanale)

Il testo dice "acetone o propanale", ma attenzione: sono due molecole diverse! L'acetone e il propan-2-one (chetone), mentre il propanale e un'aldeide. Hanno la stessa formula bruta ma struttura diversa. Risolviamo per entrambi, cosi impari a gestire queste situazioni.

Caso A: Acetone (propan-2-one)

CH3 - C(=O) - CH3

Nell'acetone abbiamo:

  • Due gruppi metilici CH3 (i carboni alle estremita)
  • Un carbonio centrale legato con doppio legame all'ossigeno (gruppo carbonilico C=O)

Assegnazione per H e O:

  • Idrogeno: n.o. = +1 (come sempre)
  • Ossigeno: n.o. = -2 (come sempre)

Calcolo n.o. dei carboni:

Carboni dei gruppi CH3 (2 carboni equivalenti):

Ciascun C del metile e legato a:

  • 3 atomi di H (legame C-H: il C "prende" l'elettrone, contributo = -1 ciascuno, totale = -3)
  • 1 atomo di C (legame C-C: nessun contributo, totale = 0)

Quindi: n.o.(C del CH3) = -3

Carbonio centrale (C=O):

Questo C e legato a:

  • 2 atomi di C (legami C-C: nessun contributo)
  • 1 atomo di O con doppio legame (2 legami con elemento piu elettronegativo: +2)

Quindi: n.o.(C centrale) = +2

Verifica: 2 x (-3) + (+2) + 6 x (+1) + (-2) = -6 + 2 + 6 - 2 = 0 ✓

Acetone (CH3-CO-CH3):
- H: +1
- O: -2
- C dei gruppi CH3: -3
- C del gruppo C=O: +2

Caso B: Propanale (aldeide)

CH3 - CH2 - CHO

Nel propanale abbiamo tre carboni diversi:

Carbonio 1 (CH3):

  • Legato a 3 H (-3) e 1 C (0)
  • n.o. = -3

Carbonio 2 (CH2):

  • Legato a 2 H (-2) e 2 C (0)
  • n.o. = -2

Carbonio 3 (CHO - gruppo aldeidico):

  • Legato a 1 H (-1), 1 C (0), e O con doppio legame (+2)
  • n.o. = -1 + 0 + 2 = +1

Verifica: (-3) + (-2) + (+1) + 6 x (+1) + (-2) = -3 - 2 + 1 + 6 - 2 = 0 ✓

Propanale (CH3-CH2-CHO):
- H: +1
- O: -2
- C del CH3: -3
- C del CH2: -2
- C del CHO: +1
3 Esercizio 2: H2O2 (acqua ossigenata)

Esercizio 2: H2O2 (acqua ossigenata)

L'acqua ossigenata e un caso speciale! E un perossido, quindi l'ossigeno NON ha il suo solito numero di ossidazione -2.

H - O - O - H

La molecola ha un legame O-O (legame perossidico). In questo legame, gli elettroni sono condivisi equamente tra i due ossigeni (stessa elettronegativita), quindi non contribuisce al n.o.

Ragionamento:

  • H ha n.o. = +1 (come sempre)
  • La molecola e neutra, quindi la somma dei n.o. deve essere 0
  • 2 x (+1) + 2 x (n.o. di O) = 0
  • 2 + 2 x (n.o. di O) = 0
  • n.o. di O = -1

Questo e tipico dei perossidi: ogni ossigeno ha n.o. = -1 perche c'e un legame O-O che non contribuisce.

H2O2 (acqua ossigenata):
- H: +1
- O: -1 (eccezione! E un perossido)
4 Esercizio 3: MgHPO4 (Idrogenofosfato di Magnesio)

Esercizio 3: MgHPO4 (Idrogenofosfato di Magnesio)

Questo e un sale ionico, quindi ragioniamo per componenti:

Mg2+ + HPO42-

Magnesio: E un metallo alcalino-terroso (Gruppo 2), quindi ha sempre n.o. = +2

Idrogeno: n.o. = +1

Ossigeno: n.o. = -2 (caso normale, non e un perossido)

Fosforo: Lo calcoliamo dalla regola che la somma deve dare la carica dello ione. L'anione e HPO42-:

(+1) + n.o.(P) + 4 x (-2) = -2

1 + n.o.(P) - 8 = -2

n.o.(P) = -2 - 1 + 8 = +5

Verifica sulla molecola neutra: (+2) + (+1) + (+5) + 4 x (-2) = 2 + 1 + 5 - 8 = 0 ✓

MgHPO4:
- Mg: +2
- H: +1
- P: +5
- O: -2
5 Esercizio 4: Acido lattico (CH3-CHOH-COOH)

Esercizio 4: Acido lattico (CH3-CHOH-COOH)

L'acido lattico ha tre atomi di carbonio diversi. Vediamoli uno per uno:

CH3 - CHOH - COOH

Per tutti:

  • H: n.o. = +1
  • O: n.o. = -2

Carbonio 1 (CH3):

  • Legato a 3 H (contributo -3) e 1 C (contributo 0)
  • n.o. = -3

Carbonio 2 (CHOH):

  • Legato a 1 H (contributo -1)
  • Legato a 2 C (contributo 0)
  • Legato a 1 O del gruppo -OH (contributo +1, legame singolo con elemento piu elettronegativo)

n.o. = -1 + 0 + 1 = 0

Carbonio 3 (COOH - gruppo carbossilico):

  • Legato a 1 C (contributo 0)
  • Legato a 1 O con doppio legame (contributo +2)
  • Legato a 1 O del gruppo -OH (contributo +1)

n.o. = 0 + 2 + 1 = +3

Verifica:

Formula molecolare: C3H6O3

Somma n.o. carboni: (-3) + (0) + (+3) = 0

Somma totale: 0 + 6 x (+1) + 3 x (-2) = 0 + 6 - 6 = 0 ✓

Acido lattico (CH3-CHOH-COOH):
- H: +1
- O: -2
- C del CH3: -3
- C del CHOH: 0
- C del COOH: +3
6 Esercizio 5: Serina (H2N-CH(CH2OH)-COOH)

Esercizio 5: Serina (H2N-CH(CH2OH)-COOH)

La serina e un amminoacido con tre atomi di carbonio. Vediamo la struttura:

H2N - CH(CH2OH) - COOH

Per tutti:

  • H: n.o. = +1
  • O: n.o. = -2
  • N: lo calcoleremo, ma in genere nei composti organici e -3 (piu elettronegativo del C, meno di O)

Azoto (N del gruppo amminico NH2):

L'azoto e legato a 2 H e 1 C. E piu elettronegativo di H e C, quindi "prende" gli elettroni da entrambi.

Calcoliamolo: nei gruppi amminici, N ha tipicamente n.o. = -3

Carbonio 1 (il C centrale, legato a NH2):

  • Legato a 1 N (contributo +1, N piu elettronegativo)
  • Legato a 1 H (contributo -1)
  • Legato a 2 C (contributo 0)

n.o. = +1 - 1 + 0 = 0

Carbonio 2 (CH2OH):

  • Legato a 2 H (contributo -2)
  • Legato a 1 C (contributo 0)
  • Legato a 1 O del gruppo -OH (contributo +1)

n.o. = -2 + 0 + 1 = -1

Carbonio 3 (COOH - gruppo carbossilico):

  • Legato a 1 C (contributo 0)
  • Legato a 1 O con doppio legame (contributo +2)
  • Legato a 1 O del gruppo -OH (contributo +1)

n.o. = 0 + 2 + 1 = +3

Verifica:

Formula molecolare: C3H7NO3

Somma: 3 C + 7 H + 1 N + 3 O

= [(0) + (-1) + (+3)] + 7 x (+1) + (-3) + 3 x (-2)

= 2 + 7 - 3 - 6 = 0 ✓

Serina (H2N-CH(CH2OH)-COOH):
- H: +1
- O: -2
- N: -3
- C centrale (legato a NH2): 0
- C del CH2OH: -1
- C del COOH: +3
7 Esercizio 6: Alanina (H3C-CH(NH2)-COOH)

Esercizio 6: Alanina (H3C-CH(NH2)-COOH)

L'alanina e un altro amminoacido, con tre carboni:

CH3 - CH(NH2) - COOH

Per tutti:

  • H: n.o. = +1
  • O: n.o. = -2
  • N: n.o. = -3 (come nei gruppi amminici)

Carbonio 1 (CH3):

  • Legato a 3 H (contributo -3)
  • Legato a 1 C (contributo 0)

n.o. = -3

Carbonio 2 (C centrale, legato a NH2):

  • Legato a 1 H (contributo -1)
  • Legato a 2 C (contributo 0)
  • Legato a 1 N del gruppo amminico (contributo +1)

n.o. = -1 + 0 + 1 = 0

Carbonio 3 (COOH - gruppo carbossilico):

  • Legato a 1 C (contributo 0)
  • Legato a 1 O con doppio legame (contributo +2)
  • Legato a 1 O del gruppo -OH (contributo +1)

n.o. = 0 + 2 + 1 = +3

Verifica:

Formula molecolare: C3H7NO2

Somma: [(-3) + (0) + (+3)] + 7 x (+1) + (-3) + 2 x (-2)

= 0 + 7 - 3 - 4 = 0 ✓

Alanina (CH3-CH(NH2)-COOH):
- H: +1
- O: -2
- N: -3
- C del CH3: -3
- C centrale (legato a NH2): 0
- C del COOH: +3

9 Concetti Chiave da Ricordare

Concetti Chiave da Ricordare

Per i composti inorganici:
  • Applica le regole standard (H = +1, O = -2, metalli con n.o. fissi)
  • Usa la regola della somma = 0 (o = carica ione) per trovare l'elemento incognito
  • Ricorda le eccezioni: perossidi (O = -1), idruri metallici (H = -1)
Per i composti organici:
  • Ogni carbonio puo avere un n.o. diverso!
  • Legame C-H: contributo -1 al C (il C "prende" l'elettrone da H)
  • Legame C-C: contributo 0 (elettronegativita uguale)
  • Legame C-O o C-N: contributo +1 (o +2 per doppio legame) al C
  • Gruppi tipici: CH3 = -3, CH2 = -2, COOH = +3, CHO = +1
