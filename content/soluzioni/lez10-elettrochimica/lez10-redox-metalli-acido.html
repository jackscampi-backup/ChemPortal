1 1. Criterio di spontaneità

ESERCIZIO 6 - Produzione di Idrogeno con Metalli in Acido

TESTO: Stabilire se è possibile produrre idrogeno aggiungendo ferro o rame metallici a una soluzione di acido cloridrico 1M. Scrivere e bilanciare la/le reazione/i che si ritiene possano avvenire.

Dati utili: E°(H⁺/H₂) = 0.00 V, E°(Fe²⁺/Fe) = -0.44 V, E°(Cu²⁺/Cu) = +0.34 V

1. Criterio di spontaneità

Una reazione redox è spontanea quando E°(cella) > 0

E°(cella) = E°(catodo) - E°(anodo)

Per produrre H₂, gli ioni H⁺ devono ridursi (catodo), mentre il metallo deve ossidarsi (anodo).

2 2. Analisi del FERRO

2. Analisi del FERRO

Semireazioni:

Riduzione: 2H⁺ + 2e⁻ → H₂     E° = 0.00 V (catodo)
Ossidazione: Fe → Fe²⁺ + 2e⁻     E° = -0.44 V (anodo)

Calcolo E°(cella):

E°(cella) = E°(catodo) - E°(anodo)
E°(cella) = 0.00 - (-0.44) = +0.44 V
E°(cella) = +0.44 V > 0 → REAZIONE SPONTANEA!
Il ferro reagisce con HCl producendo idrogeno.
3 3. Analisi del RAME

3. Analisi del RAME

Semireazioni (ipotetiche):

Riduzione: 2H⁺ + 2e⁻ → H₂     E° = 0.00 V
Ossidazione: Cu → Cu²⁺ + 2e⁻     E° = +0.34 V

Calcolo E°(cella):

E°(cella) = 0.00 - (+0.34) = -0.34 V
E°(cella) = -0.34 V < 0 → REAZIONE NON SPONTANEA!
Il rame NON reagisce con HCl.
4 4. Bilanciamento della reazione che avviene

4. Bilanciamento della reazione che avviene

Solo il ferro reagisce. Bilanciamo:

Reazione ionica: Fe + 2H⁺ → Fe²⁺ + H₂

Reazione molecolare: Fe + 2HCl → FeCl₂ + H₂

Verifica: Fe: 1=1 ✓, H: 2=2 ✓, Cl: 2=2 ✓

5 5. Perché questa differenza?

5. Perché questa differenza?

MetalloE° (V)E°(cella) (V)Reagisce?
Ferro-0.44+0.44SÌ
Rame+0.34-0.34NO

Regola: Un metallo può spostare H₂ da un acido solo se ha E° < 0 (potenziale minore di quello dell'idrogeno).

  • Il ferro (E° = -0.44 V < 0) è meno nobile dell'idrogeno → reagisce
  • Il rame (E° = +0.34 V > 0) è più nobile dell'idrogeno → non reagisce
RISPOSTA: È possibile produrre idrogeno solo con il FERRO.
Reazione: Fe + 2HCl → FeCl₂ + H₂
