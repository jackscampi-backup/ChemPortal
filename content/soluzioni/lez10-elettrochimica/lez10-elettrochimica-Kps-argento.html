1 Introduzione: cos'e la Kps e perche usare l'elettr...

Determinazione della Kps per via elettrochimica

QUESITO: Si vuole determinare la Kps di un sale d'argento poco solubile per via elettrochimica. Disegnare il dispositivo, e indicare le formule per calcolare il valore richiesto.

Introduzione: cos'e la Kps e perche usare l'elettrochimica?

Prima di tutto, fermiamoci a capire cosa stiamo cercando di fare. La Kps (costante del prodotto di solubilita) e un valore che ci dice quanto e solubile un sale poco solubile. Per un sale come AgCl, la reazione di dissoluzione e:

AgCl(s) ⇌ Ag+(aq) + Cl-(aq)

E la Kps e definita come:

Kps = [Ag+] × [Cl-]

Il problema e: come misuriamo la Kps? Per sali molto poco solubili (come AgCl, con Kps nell'ordine di 10^-10), la concentrazione degli ioni in soluzione e talmente bassa che i metodi tradizionali (pesata, titolazione) diventano imprecisi.

Idea chiave: Possiamo usare l'elettrochimica perche il potenziale di un elettrodo dipende dalla concentrazione degli ioni in soluzione. Misurando una differenza di potenziale (f.e.m.), possiamo risalire alle concentrazioni e quindi alla Kps!
2 Il dispositivo: una pila a concentrazione

Il dispositivo: una pila a concentrazione

Per determinare la Kps di un sale d'argento (prendiamo AgCl come esempio), costruiamo una pila a concentrazione con due semicelle contenenti entrambe elettrodi d'argento, ma con concentrazioni diverse di ioni Ag+.

Schema del dispositivo

         ANODO (-)                              CATODO (+)
    [Ag+ a bassa conc.]                    [Ag+ a conc. nota]

     _______________                        _______________
    |               |                      |               |
    |   Elettrodo   |                      |   Elettrodo   |
    |      Ag       |                      |      Ag       |
    |       |       |                      |       |       |
    |       |       |      VOLTMETRO       |       |       |
    |       |       |         ___          |       |       |
    |       |_______|________|   |_________|_______|       |
    |               |        |_V_|         |               |
    |               |                      |               |
    |   Soluzione   |     Ponte salino     |   Soluzione   |
    |   satura di   |      =========       |   AgNO3       |
    |   AgCl + NaCl |                      |   1 M         |
    |   (Cl- 1 M)   |                      |   [Ag+]=1 M   |
    |_______________|                      |_______________|

         E1                                      E2
    (semicella con                         (semicella di
     AgCl saturo)                          riferimento)

Descrizione delle due semicelle

Semicella SINISTRA (Anodo) Semicella DESTRA (Catodo)
Elettrodo di Ag immerso in una soluzione satura di AgCl con eccesso di Cl- (es. NaCl 1 M) Elettrodo di Ag immerso in soluzione di AgNO3 a concentrazione nota (es. 1 M)
[Ag+] = incognita (molto bassa, dell'ordine di 10^-10 M) [Ag+] = 1 M (concentrazione nota)
E1 = E° + (0,059/1) × log[Ag+]bassa E2 = E° + (0,059/1) × log[Ag+]alta
3 Perche funziona questo dispositivo?

Perche funziona questo dispositivo?

Ti starai chiedendo: perche usiamo questa configurazione particolare? Ragionamo insieme.

Nella semicella di sinistra, abbiamo una soluzione satura di AgCl in presenza di ioni Cl- in eccesso. Cosa succede qui? Il sale AgCl e poco solubile, quindi si stabilisce l'equilibrio:

AgCl(s) ⇌ Ag+(aq) + Cl-(aq) con Kps = [Ag+] × [Cl-]

Se aggiungiamo un eccesso di Cl- (per esempio usando NaCl 1 M), per l'effetto dello ione comune, l'equilibrio si sposta verso sinistra e la concentrazione di Ag+ diventa estremamente bassa. Quanto bassa? Possiamo calcolarla:

[Ag+] = Kps / [Cl-]

Se [Cl-] = 1 M, allora [Ag+] = Kps (numericamente).

Nella semicella di destra, abbiamo una soluzione di AgNO3 a concentrazione nota (tipicamente 1 M). L'AgNO3 e un sale completamente solubile, quindi [Ag+] = 1 M.

Il trucco: Le due semicelle hanno la stessa reazione redox (Ag+ + e- ⇌ Ag), ma concentrazioni di Ag+ molto diverse. Questa differenza di concentrazione genera una forza elettromotrice misurabile!
4 Le formule per il calcolo

Le formule per il calcolo

Passo 1: Equazione di Nernst per ogni semicella

Per ogni semicella con elettrodo di argento, il potenziale e dato dall'equazione di Nernst:

E = E°(Ag+/Ag) + (0,059 / n) × log[Ag+]

dove:

  • E°(Ag+/Ag) = +0,80 V (potenziale standard della coppia Ag+/Ag)
  • n = 1 (numero di elettroni scambiati)
  • [Ag+] = concentrazione degli ioni argento

Semicella destra (catodo, concentrazione nota):

E2 = E°(Ag+/Ag) + 0,059 × log[Ag+]nota

Se [Ag+]nota = 1 M, allora log(1) = 0, quindi E2 = E° = +0,80 V

Semicella sinistra (anodo, soluzione satura):

E1 = E°(Ag+/Ag) + 0,059 × log[Ag+]satura

Passo 2: Calcolo della f.e.m. della pila

La forza elettromotrice (f.e.m.) della pila e:

f.e.m. = E(catodo) - E(anodo) = E2 - E1

Sostituendo le espressioni di Nernst:

f.e.m. = [E° + 0,059 × log[Ag+]nota] - [E° + 0,059 × log[Ag+]satura]

Nota che E° si cancella! Questo e uno dei vantaggi della pila a concentrazione:

f.e.m. = 0,059 × (log[Ag+]nota - log[Ag+]satura)

Usando le proprieta dei logaritmi (log A - log B = log(A/B)):

f.e.m. = 0,059 × log([Ag+]nota / [Ag+]satura)

Passo 3: Dalla f.e.m. alla concentrazione di Ag+ nella soluzione satura

Riarrangiando la formula precedente per ricavare [Ag+]satura:

log([Ag+]nota / [Ag+]satura) = f.e.m. / 0,059
[Ag+]nota / [Ag+]satura = 10^(f.e.m. / 0,059)
[Ag+]satura = [Ag+]nota / 10^(f.e.m. / 0,059)

Oppure, in modo equivalente:

[Ag+]satura = [Ag+]nota × 10^(-f.e.m. / 0,059)

Passo 4: Calcolo della Kps

Una volta trovata [Ag+]satura, possiamo calcolare la Kps. Ricorda che nella semicella di sinistra abbiamo:

Kps = [Ag+]satura × [Cl-]

Se abbiamo usato NaCl 1 M per avere [Cl-] = 1 M, allora:

Kps = [Ag+]satura × 1 = [Ag+]satura
FORMULA FINALE:

Kps = [Ag+]nota × [Cl-] × 10^(-f.e.m. / 0,059)

Se [Ag+]nota = 1 M e [Cl-] = 1 M:

Kps = 10^(-f.e.m. / 0,059)
5 Esempio numerico

Esempio numerico

Supponiamo di misurare una f.e.m. = 0,59 V. Quanto vale la Kps?

Kps = 10^(-0,59 / 0,059) = 10^(-10) = 1 × 10^(-10)

Questo valore e coerente con la Kps nota di AgCl (circa 1,8 × 10^-10 a 25°C).

6 Notazione della pila

Notazione della pila

La pila puo essere rappresentata con la notazione convenzionale:

Ag | AgCl(sat), Cl-(1 M) || Ag+(1 M) | Ag

Dove:

  • La barra singola | indica l'interfaccia elettrodo/soluzione
  • La doppia barra || indica il ponte salino
  • L'anodo (ossidazione) e a sinistra, il catodo (riduzione) a destra
7 Riepilogo del metodo

Riepilogo del metodo

Procedura sperimentale:
  1. Costruire la pila con le due semicelle come descritto
  2. Collegare un voltmetro ad alta impedenza e misurare la f.e.m.
  3. Applicare la formula: Kps = [Ag+]nota × [Cl-] × 10^(-f.e.m. / 0,059)
VANTAGGI DEL METODO ELETTROCHIMICO:
  • Permette di misurare Kps molto piccole (anche 10^-18)
  • E' un metodo rapido e preciso
  • Non richiede di misurare direttamente concentrazioni bassissime
  • Il potenziale standard si cancella, riducendo gli errori sistematici
